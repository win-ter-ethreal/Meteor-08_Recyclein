## 📌 Deskripsi
Proyek ini dibuat untuk mata kuliah **Manajemen Proyek** dengan tema aplikasi berbasis web untuk penjemputan sampah.  
Tujuan utama: mempermudah masyarakat menjadwalkan penjemputan sampah sekaligus memberikan sistem poin & reward.

## ✨ Fitur Utama
- Registrasi & Login User
- Dashboard User
- Reservasi Jadwal Penjemputan
- Katalog Sampah & Poin
- Redeem Reward (Voucher/Saldo)
- Admin Panel (Kelola User, Jadwal, Reward)
